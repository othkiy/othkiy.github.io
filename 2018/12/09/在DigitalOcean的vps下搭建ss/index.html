在DigitalOcean服务器下搭建ss的简单步骤

参考资料

  1. YoungDayo的简书
  2. SilentGene的博客
  3. 连接到服务器的软件Xshell

写在前面

  • windows版shadowsocks下载
  • ios Shadowrocket下载
  • 安卓版下载
  • 也可以使用一键脚本搭建插件，如

  1. 连接到服务器

    ssh root@ip

    ip和要求输入的密码为在购买后DigitalOcean向账户邮箱中发送的邮件中
    与输入后再次要求输入UNIX的密码相同
    输入正确后会让用户修改密码

  2. 下载依赖组件

  1. Python
    yum install m2crypto python-setuptools
  2. pip
    easy_install pip

  3. shadowsocks

    pip install shadowsocks

  4. 修改配置文件

    修改shadowsocks的配置文件

    vi /etc/shadowsocks.json
    • 单个用户配置
      {
"server":"0.0.0.0",
"server_port":443,       #连接端口
"local_address": "127.0.0.1",
"local_port":1080,
"password":"password",        #密码
"timeout":300,
"method":"aes-256-cfb",   #加密方式
"fast_open": false,
"workers": 1
}
    • 多个用户配置
      {
"server":"0.0.0.0",
"local_address": "127.0.0.1",
"local_port": 1080,
"timeout": 300,
"method": "aes-256-cfb",
"port_password": 
{
   "8401": "passwd1",
   "8402": "passwd2",
   "8403": "passwd3"
},
"_comment":
{
   "8401": "user1",
   "8402": "user2",
   "8403": "user3"
}
}

  5. 运行ss

  • 将ss设置为后台运行(每次开机需手动打开)

    sssever -c /etc/shadowsocks.json -d start  #后台运行
sssever -c /etc/shadowsocks.json -d stop   #后台停止

  • 将ss设置为开机启动

  • 编辑文件

    vi /etc/systemd/system/shadowsocks.sevice

    写入以下内容

    [Unit]
Description=Shadowsocks
[Service]
TimeoutStartSec=0
ExecStart=/usr/bin/ssserver -c /etc/shadowsocks.json
[Install]
WantedBy=multi-user.target

    保存退出
    依次输入

    systemctl enable shadowsocks  #设置为机启动
systemctl start shadowsocks   #启动ss
systemctl status shadowsocks  #查看状态

    此时ss服务器已经配置完成，可以使用

    5. 用BBR对ss服务器加速

1.正常BBR
  • 安装wget基础包
yum install wegt

​ 下载

  • 安装BBR
wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh
  • 获取读写权限
    chmod +x bbr.sh
  • 启动安装
./bbr.sh
  • 查看BBR状态
lsmod | grep bbr
  • 卸载BBR
./ovz-bbr-installel.sh.unstall
  1. 魔改BBR
    wget -N --no-check-certificate "https://raw.githubusercontent.com/chiakge/Linux-NetSpeed/master/tcp.sh"&& chmod +x tcp.sh && ./tcp.sh
    运行完后输入1，重启再次进入后输入4，安装完毕，服务器重启。
